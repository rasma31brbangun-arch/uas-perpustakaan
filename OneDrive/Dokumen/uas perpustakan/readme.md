# Sistem Perpustakaan Sederhana (UAS)

## Deskripsi
Aplikasi Sistem Informasi Perpustakaan berbasis Command Line Interface (CLI) menggunakan Python untuk memenuhi syarat Ujian Akhir Semester (UAS).

## Fitur Aplikasi
1. Login Sederhana (Username: admin | Password: admin)
2. CRUD Data Buku (Create, Read, Update, Delete)
3. Pencarian Data Buku

## Cara Menjalankan
Buka terminal dan jalankan perintah: `python main.py`
