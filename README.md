# Sistem Perpustakaan Sederhana (UAS)

## 1. Nama Proyek
Sistem Informasi Perpustakaan Berbasis CLI (Command Line Interface).

## 2. Deskripsi
Aplikasi ini dibuat menggunakan bahasa pemrograman Python untuk memenuhi tugas Ujian Akhir Semester (UAS). Aplikasi ini dilengkapi dengan sistem Login sederhana, fitur CRUD (Tambah, Lihat, Edit, Hapus) data buku, dan pencarian buku.

## 3. Cara Instalasi
Pastikan komputer atau laptop sudah terinstal Python versi 3.x. Tidak ada *library* tambahan yang perlu diinstal karena aplikasi ini menggunakan bawaan standar Python.

## 4. Cara Menjalankan
1. Buka Terminal atau Command Prompt.
2. Arahkan ke folder proyek ini.
3. Ketik perintah berikut lalu tekan Enter:
   `python main.py`
4. Gunakan kredensial berikut untuk **Login**:
   - Username: **admin**
   - Password: **admin**

## 5. Struktur Folder
- `main.py` : Berisi kode logika utama program perpustakaan.
- `README.md` : Berisi dokumentasi dan panduan proyek ini.

## 6. Riwayat Commit
Proyek ini telah menerapkan sistem *Version Control* (Git) dengan tahapan *branching*, *merge*, penyelesaian konflik, serta pembuatan *tag release* (v1.0.0) sesuai syarat UAS.
